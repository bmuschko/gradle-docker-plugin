            • void addFile(String src, String dest)

              The ADD instruction copies new files, directories or remote file URLs from and adds them to the filesystem of the container at the path . Example in Groovy DSL: 

               task createDockerfile(type: Dockerfile) {
     addFile('test', '/absoluteDir/')
 }
              Parameters:
              src - Source file
              dest - Destination path
              See Also:
